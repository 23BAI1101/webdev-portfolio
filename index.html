Basketball Shot Detection

This project detects baskets for playing basketball from various distances and angles. It was trained on about 5000 images using Roboflow , Yolov8, PyTorch and Tensorflow. This will be used in the upcoming Robocon competition to be held in IIT Delhi.

RFID Attendance

We made use of ESP8266 to send messages using google spreadsheets , with date time and name whenever a student would punch the card. This project was utilized in a lab where it kept a track of how many components each team of students takes . The same procedure was followed while returning the components.We used an rfid module , LCD screen with I2C module and led lights .

Anomaly Detection System

This project made use of the real-time data in the system, all system metrics like hard disk space ,speed etc. A fresh dataset of 100 records were made each time we wish to implement this system. Then after getting trained we could test it on real-time data itself.If any anomaly (deviation from the trained data) was detected it could be sent to the email of the user .
